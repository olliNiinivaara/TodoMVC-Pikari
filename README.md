# TodoMVC-Pikari
The classic [TodoMVC](http://todomvc.com/examples/vanilla-es6/) on [Pikari](https://github.com/olliNiinivaara/Pikari/).

This application is testable on-line *as-is* at <https://verkkoyhteys.fi/pikari/todomvc>
